# NH Multilingual Learners Analysis

## Research Purpose
This project explores the distribution and educational outcomes of multilingual learners (MLs) across New Hampshire school districts.

## Research Questions
1. How are multilingual learners distributed across New Hampshire districts?
2. Is there a relationship between multilingual learner population and academic performance?
3. How have multilingual learner populations changed over time?

## Data Source
- New Hampshire Department of Education public data
- NCES district-level education data (when available)

## Methods
- Data cleaning in R
- Descriptive statistics
- Data visualization
- Correlation and regression analysis

## Tools Used
- R (tidyverse, ggplot2)
- GitHub for version control
